Introduction

This project is demonstration of usage of markup languages. It involves a small fraction of patients from veterinary clinic(pets) and their respectable owners. This sample consists of basic but useful information about pets and the owners.

These are files that are part of this solution
animals.xml
Structured and validated XML document with all the information
animals.xsd
XSD document which defines the XML structure
animals.xslt
XSLT document prescription for XML document
animals.html
Result of XSL transformation which shows table with basic information
animalsdocumentation.html
Project documentation in html

Methodology

For this project were used some of free markup language validation and generator tools listed here.

  • XSD and XML document validator
  • XSD Schema generator
  • Transformation tool for XSL

Element and Attribute description

Elements are split into multiple sections"

AnimalInfo

This section consists of all the information regarding the pet except vaccinations

Gender
Animal gender
Name
Animal's name
AnimalType
Describes what animal it is (dog,cat...etc.)
Breed
Breed of the animal
CountryOfOrigin
From which country the animal comes from
DateOfBirth
Animal's date of birth
Atributes formate describes in which way is the date formated(YYYY-MM-DD)
WeightKG
Animal's weight in kilograms
HeightCM
Animal's height in centimeters
Coloring
Color of animal's fur or skin
VaccinationID
ID number of vaccination
EnteredSystem
Date when the animal was added into the system
Atributes formate describes in which way is the date formated(YYYY-MM-DD)
Changes
Date when the last change was made to the animal information
Atributes formate describes in which way is the date formated(YYYY-MM-DD)
LastVisit
Date of the last visit with the given animal
Atributes formate describes in which way is the date formated(YYYY-MM-DD)

OwnerInfo

Section with animal's owner identification

Name
Name of the owner
Surname
Surname of the owner
Phone
Owner's phone number
Atributes type describes what type of phone it is
Email
Animal owner's email address
Fax
Animal's owner fax number

OwnerAdress

Animal owner's adress

Street
Street where the owner lives
Number
Number of house
Atributy: adress house number
City
City where the owner lives
CityPart
Part of the city where the owner lives
ZIP
ZIP code of the area
State
State where the city is located
StateRegion
Region of state where the city is located

LastVaccination

This section contains only dates when the last vaccination has been done and therefore only one description is needed since all are the same

Parvovirus
Date of the last Parvovirus vaccination
Atributes formate describes in which way is the date formated(YYYY-MM-DD)

Transformation description

Transformation of XML file to HTML done by site mentioned in methodology. Result is a table with very basic information such as owner's complete name, which animal he owns, what breed it is, gender of the animal, date of birth and lastly phone number for the owner.
